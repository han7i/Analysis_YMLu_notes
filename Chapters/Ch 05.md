### 5.4 无穷小量与无穷大量的阶

> 7. 用 $\pi(x)$ 表示不超过 $x$ 的素数的个数，若已知 \[
 \pi(x) \sim \frac{x}{\ln x}, \quad x \to +\infty;
\] 证明：第 $n$ 个素数 $p_n$ 满足 $p_n \sim n \ln n, n \to +\infty.$  

$n$ 趋于 $+\infty$ 时有 \[
 \frac{p_n}{n \ln n} = \frac{p_n}{\pi(p_n) \ln \pi(p_n)} = \frac{\ln p_n}{\ln p_n - \ln \ln p_n} = 1.
\] 

> 8. 证明对任意 $x > 1$ 有 \[
 \sum^\infty_{n=1} \frac{1}{x+2^n} = \frac{\log_2 x}{x} + O\left(\frac{1}{x}\right)
\] 

记 $N = \lfloor \log_2 x \rfloor,$ 将原级数分解为如下部分和与余和： \[
 S_1 = \sum^{N}_{n=1} \frac{1}{x+2^n}, \quad S_2 = \sum^\infty_{n=N+1} \frac{1}{x+2^n},
\] 对于 $S_1$ 有 \[
 \sum^N_{n=1} \frac{1}{x+2^n} = \sum^N_{n=1} \left( \frac{1}{x} - \frac{2^n}{x(x+2^n)} \right) = \frac{N}{x} + \sum^N_{n=1}\frac{2^n}{x(x+2^n)},
\] 由于 $2^n \leq x,$ 于是其中 \[
 \sum^N_{n=1}\frac{2^n}{x(x+2^n)} < \sum^N_{n=1} \frac{2^n}{x^2} < \frac{2^{N+1}}{x^2} < \frac{2}{x},
\] 又 $N = \log_2 x + O(1),$ 故 \[
 S_1 = \frac{\log_2 x}{x} + O\left(\frac{1}{x}\right);
\] 
对于 $S_2$ 有 \[
 \sum^\infty_{n=N+1} \frac{1}{x+2^n} < \sum^\infty_{n=N+1} \frac{1}{2^n} = \frac{1}{2^N} < \frac{2}{x},
\] 综上即得。 

### 5.5 连续函数

> 15. 证明：定义在 $\mathbb{R}$ 上的非常值连续周期函数必有最小正周期。

设函数 $f$ 满足题设，记 $C(f)$ 为其正周期之集合。由 $C(f)$ 非空且 $0$ 为其下界知 $\inf C(f)$ 存在。假设 $\inf C(f) \notin C(f),$ 则存在一列正周期 $\{ c_n \}$ 满足 $\displaystyle \lim_{n \to \infty} c_n = \inf C(f),$ 而两周期之差仍为原函数的一个周期，由柯西收敛定理知存在任意小的正周期，即 $\inf C(f) = 0;$
\[
 \forall x \in \mathbb{R}\, \forall n \in \mathbb{Z}_{>0}, x - c_n \leq \left\lfloor \frac{x}{c_n} \right\rfloor c_n < x,
\] 因而 \[
 f(x) = f(\lim_{n \to \infty}\left\lfloor \frac{x}{c_n} \right\rfloor c_n) = \lim_{n \to \infty}f(0+\left\lfloor \frac{x}{c_n} \right\rfloor c_n) = f(0),
\] 与 $f(x)$ 非常值矛盾。故 $\inf C(f) \in C(f)$ 即为最小正周期。

（可以证明任意 $f$ 的全部周期构成 $(\mathbb{R}, +)$ 的子群，因而要么为离散子群 $a\mathbb{Z} (a \geq 0),$ 要么稠密子群；加上连续性限制后进一步成为闭子群，$\mathbb{R}$ 的稠密闭子集只有其本身，即对应常值函数。参考 MathABC 9.2 习题 9.14.）

> 16. 设 $f$ 是定义在 $(a,b)$ 上的有界函数，记 \[
 \alpha(x) = \inf_{t \in (a,x)}f(t), \quad \beta(x) = \sup_{t \in (a,x)}f(t). 
\] 
>> (1) 证明 $\alpha, \beta$ 在 $(a,b)$ 上左连续，但未必右连续。

显然 $\alpha$ 单调递减，因而左右极限存在。考虑在 $x_0 \in (a,b)$ 处的连续性： 
由单调性有 $\alpha(x_0^-) \geq \alpha(x_0);$
又由下确界定义知，$\forall \varepsilon > 0\, \exist\delta > 0, \alpha(x_0) + \varepsilon > f(x_0 - \delta),$ 考虑 $w \in (x_0 - \delta, x_0),$ 有 $\alpha(w) \leq f(x_0 - \delta) < \alpha(x_0) + \varepsilon,$ 令 $\delta \to 0^+$ 并由 $\varepsilon$ 的任意性即得 $\alpha(x_0^-) \leq \alpha(x_0).$
综上，$\alpha(x_0^-) = \alpha(x_0).$ 对 $\beta$ 证明类似。
由于 $x_0 + \delta \notin (a,x_0),$ 上述证明不适用于右连续。一个简单的反例是 $f(x) = - \lfloor x \rfloor.$

>> (2) 若 $f$ 为连续函数，证明 $\alpha, \beta$ 亦然。

由单调性有 $\alpha(x_0^+) \leq \alpha(x_0);$
由连续性知 $\forall \varepsilon > 0\, \exist \delta > 0, \left\lvert f(x_0) - f(x_0 + \delta) \right\rvert < \varepsilon,$ 由单调性，即 $f(x_0) < f(x_0 + \delta) + \varepsilon.$ 
又 $\forall v \in (a, x_0), f(v) \geq \alpha(x_0),$ 由连续性，令 $v$ 趋于 $x_0^-$ 得 $f(x_0) = f(x_0^-) \geq \alpha(x_0),$ 故 $f(x_0 + \delta) > \alpha(x_0) - \varepsilon,$ 因而 \[\begin{aligned}
 \alpha(x_0 + \delta) &= \inf_{t \in (a, x_0 + \delta)} f(t)  = \min \left(\alpha(x_0), \inf_{t \in [x_0, x_0 + \delta)} f(t) \right), \\
 &\geq \min(\alpha(x_0), \alpha(x_0) - \varepsilon) = \alpha(x_0) - \varepsilon,
\end{aligned}\] 即 $\alpha(x_0) - \varepsilon \leq \alpha(x_0^+) \leq \alpha(x_0),$ 由 $\varepsilon$ 的任意性即得。  
 
> 17. 设 $\sigma:\mathbb{Z}_{>0}\to\mathbb{Q}$ 为双射，记 $r_n = \sigma(n)$，证明 \[
 f:\mathbb{R}\to\mathbb{R}, \quad x\mapsto \sum_{\begin{subarray}{c}
   n\\
   r_n < x
\end{subarray}} \frac{1}{2^n}
\] 为在无理点连续、在有理点间断的严格单调递增函数。

由 $\mathbb{Q}$ 的稠密性知 $f$ 严格单调递增。
在无理点 $x_{ir}$ 处，对任意 $\varepsilon > 0,$ 取 $N$ 使得余和 $\displaystyle \sum^\infty_{n=N+1} \frac{1}{2^n} = \frac{1}{2^N} < \varepsilon,$ 于是只需要取 $\delta$ 使得 $\delta$-邻域不含有限项的 $r_1, r_2, \ldots r_N$ 即有 $\forall x \in (x_{ir} - \delta, x_{ir} + \delta), \left\lvert f(x_{ir}) - f(x) \right\rvert < \varepsilon,$ 因而 $f$ 在 $x_{ir}$ 处连续。
在有理点 $x_r$ 处，显然 $\displaystyle f(x_r^+) - f(x_r) = \frac{1}{2^{\sigma^{-1}(x_r)}},$ 因而 $f$ 在 $x_r$ 处间断。

### 5.7 闭区间上连续函数的性质

> 5. 设 $f$ 是 $[a,b]$ 上的连续函数，且 $f(0) = f(1).$ 证明：对任意的 $n \in \mathbb{Z}_{>0},$ 均存在 $\xi_n \in [0,1]$ 使得 $\displaystyle f(\xi_n + \frac{1}{n}) = f(\xi_n).$

记 $\displaystyle F_n(x) = f(x + \frac{1}{n}) - f(x),$ $F_n(x)$ 的零点即可取为 $\xi_n.$ 由 \[
 F_n(0) + F_n(\frac{1}{n}) + \cdots + F_n(\frac{n-1}{n}) = f(1) - f(0) = 0,
\] 若左式 $n$ 项均为 $0$ 则其均满足要求，否则必然存在两项符号相异，由零点定理即得。

> 9. 用有限覆盖定理证明零点定理。

反设 $f(x)$ 在 $(a,b)$ 上没有零点。考虑开区间族 $\mathcal{L} = \{ l \mid \forall x \in l, f(x) < 0 \}$ 及 $ \mathcal{U} = \{ u \mid \forall x \in u, f(x) > 0 \},$ 由连续性知 $[a,b]$ 上任意一点均存在 $\varepsilon$-邻域保持其中 $f(x)$ 符号不变，因而 $\mathcal{L} \cup \mathcal{U}$ 构成 $[a,b]$ 的一族开覆盖，由有限覆盖定理知存在一组有限开覆盖 $L \cup U,$ 由于 $f(a), f(b)$ 异号，$L, U$ 均非空，故与连通性矛盾（或用三歧性）。

### 5.8 一致连续性

> 3. 设函数 $f(x)$ 在 $[a,b)$ 上连续。证明：$f(x)$ 在 $[a,b)$ 上一致连续的充要条件是 $\displaystyle \lim_{x \to b^-} f(x)$ 存在。

$(\Rightarrow)$ $\forall x_1, x_2 \in (b - \delta, b), |f(x_1) - f(x_2)| < \varepsilon,$ 因而趋于 $b^-$ 的数列均为柯西列，由柯西收敛定理知其收敛，又由海涅定理知右极限存在。

$(\Leftarrow)$ 设 $\displaystyle \lim_{x \to b^-} f(x) = B,$ 定义 $\widetilde{f}(b) = B$ 将 $f(x)$ 延拓到 $[a,b]$ 上，由海涅-康托尔定理，$\widetilde{f}(x)$ 在 $[a,b]$ 上一致连续，于是在 $[a,b)$ 上 $f(x) = \widetilde{f}(x)$ 一致连续。

> 6. 设函数 $f(x)$ 在 $[a,+\infty)$ 上连续。证明：若 $\displaystyle \lim_{x \to +\infty} f(x)$ 存在，则 $f(x)$ 在 $[a,+\infty)$ 上一致连续。

设 $\displaystyle \lim_{x \to +\infty} f(x) = B.$ 在 $[a,N_\varepsilon]$ 上用海涅-康托尔定理可知 $f(x)$ 一致连续，而在 $(N_\varepsilon, +\infty)$ 上有 $|x_1 - x_2| \leq |x_1 - B| + |x_2 - B| < 2\varepsilon.$
逆命题的典例如 $f(x) = x$ 或 $\sin x.$

（对比 3.， $[a,b)$ 完备化即得到 $[a,b],$ 而 $[a,+\infty)$ 已经是完备的。）

> 8. 利用波尔查诺-魏尔斯特拉斯定理证明海涅-康托尔定理。

反设 $f(x)$ 在 $[a,b]$ 不一致连续，则存在数列 $\{ a_n \}, \{ b_n \} \subseteq [a,b]$ 满足 $a_n - b_n \to 0 \, (n \to +\infty)$ 且 $|f(a_n) - f(b_n)| \geq \varepsilon_0.$
显然 $\{ b_n \}$ 有界，由波尔查诺-魏尔斯特拉斯定理知存在子列 $\{ b_{n_i} \}$ 收敛于 $\xi \in [a,b],$ 则 $\{ a_{n_i} \} = \{ (a_{n_i} - b_{n_i}) + b_{n_i} \}$ 收敛于 $0 + \xi = \xi,$ 由连续性，$|f(a_{n_i}) - f(b_{n_i})| \to |f(\xi) - f(\xi)| = 0 \, (i \to +\infty),$ 与其大于 $\varepsilon_0$ 矛盾。 

> 9. 设 $f(x)$ 在 $[0,+\infty)$ 上一致连续，证明 $f(x) = O(x) \, (x \to +\infty).$

\[
 \forall x \in [0,+\infty), |f(x) - f(0)| \leq \varepsilon \left\lceil \frac{x}{\delta} \right\rceil.

> (梅 3.4.2 17.) 证明连续的周期函数是一致连续的。

设 $f(x)$ 满足题设，$T$ 为其一个周期，对任意 $\varepsilon,$ $f(x)$ 在 $[0,T]$ 上一致连续，有对应的 $\delta_1,$ 在 $\displaystyle [\frac{1}{2}T, \frac{3}{2}T]$ 上同样一致连续，有 $\delta_2;$ 取 $\displaystyle \delta = \min(\delta_1, \delta_2, \frac{T}{4}),$ 于是对于任意 $|x - y| \leq \delta,$ 其总落在上述两种区间平移 $n$ 个周期所得区间之中。

### 5.9 无穷乘积

> \[
 (1-x)\prod_{k=1}^{N}(1+x^{2^{k-1}}) = (1-x^2)\prod_{k=2}^{N}(1+x^{2^{k-1}}) = \cdots = 1 - x^{2^N} \\
 \prod_{k=1}^{N}(1+x^{2^{k-1}}) = \frac{1 - x^{2^N}}{1-x} = \sum^{2^N}_{n=1} x^{n-1} \\
 \prod_{k=1}^{\infty}(1+x^{2^{k-1}}) = \sum^{\infty}_{n=1} x^{n-1},
\] 右式中 $x$ 的指数遍历了所有自然数，且各项系数为 $1$ ，因而左式中 $x$ 的指数 $a_0 2^0 + a_1 2^1 + \cdots (a_i \in \{ 0,1 \})$ 亦将唯一地遍历这些自然数。
二进制中，$k$ 位可选 $0,1,$ 对应 $(1 + x^{2^k}),$ 对于 $p$-进制，则对应 \[
 1+x^{p^k}+x^{2p^k}+\cdots+x^{(p-1)p^k},
\] 有 \[
 \prod_{k=0}^{N-1}\left(\sum_{a=0}^{p-1}x^{a\cdot p^k}\right) = \prod_{k=0}^{N-1}\frac{1-x^{p^{k+1}}}{1-x^{p^k}} = \frac{1-x^{p^N}}{1-x},
\] 或说“$N-1$ 位均为 $p-1$ 时，加 $1$ 将导致进位得到 $N$ 位为 $1$ 其余为 $0$.” 即通过裂项相消与等比数列求和复刻了 $p$-进制下的进位规则。
令 $N$ 趋于无穷即得 \[
 \prod_{k=0}^{\infty}\left(\sum_{a=0}^{p-1}x^{a\cdot p^k}\right) = \sum_{m=0}^{\infty}x^m,
\] 或说任意自然数都有唯一的 $p$-进制表示。

> （欧拉整数划分定理）对于任意整数，将其划分为互不相同整数之和与将其划分为奇数之和的方式数量相同。
> \[
 1 + x^k = \frac{1-x^{2k}}{1-x^k} \\
 \prod_{k=1}^{\infty}(1+x^k) = \prod_{k=1}^{\infty}\frac{1}{1-x^{2k-1}}.
\] 左式对应不同整数，右式对应奇数。
