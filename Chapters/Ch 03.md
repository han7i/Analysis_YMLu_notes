## 数列的极限

### 3.1 数列极限的定义

> 11. （柯西极限定理）设 $\lim_{n \to \infty}x_n = a$，证明 \[\lim_{n \to \infty}\frac{x_1+x_2+ \cdots + x_n}{n} = a\].

由 $\{ a_n \}$ 收敛，$\forall \varepsilon>0\,\exist n_{\varepsilon}\,\forall n\geq n_{\varepsilon}, |x_n-a| < \varepsilon $，则 $n>k$ 时有

\[ \begin{aligned}
 \left\lvert \frac{1}{n}\sum^n_{k=1}x_k - a \right\rvert  &= \left\lvert \frac{1}{n}\left( \sum^{n_\varepsilon-1}_{j=1}x_j + \sum^n_{k=n_\varepsilon}x_k -na \right) \right\rvert \\
 &\leq \left\lvert \frac{1}{n}\left( \sum^{n_\varepsilon-1}_{j=1}(x_j-a) \right) \right\rvert + \left\lvert \frac{n-n_\varepsilon +1}{n}\varepsilon \right\rvert,
\end{aligned}\]

令上式小于 $2\varepsilon$，取 $n>\left\lceil \frac{\sum^{n_\varepsilon-1}_{j=1}(x_j-a)}{\varepsilon} \right\rceil $ 即可。

（部分和算术平均称为切萨罗和（Cesàro summation），其值收敛则称切萨罗可和。上述命题逆命题不成立，典型例子如震荡序列，需要添加额外限制，例如序列单调。一函数的傅里叶级数不一定收敛，但在连续点处切萨罗可和且值等于函数值。）

> 12. 若 $\{ x_n \}$ 收敛，证明在该数列中一定有最大数或最小数，但不一定同时有最大数和最小数。

假设 $\{ x_n \}$ 收敛于 $a$，若 $\{ x_n \} = \{ a \}$ 命题显然成立，否则取使得下标最小的 $x_i \neq a$，令 $\varepsilon = |x_i - a| > 0$，则有限集 $\{ x_i, \ldots ,x_{n_\varepsilon - 1} \}$ 的最大数为数列的最大数，或最小数为数列的最小数，两者可能同时成立。

> 14. 设 $\varphi$ 是 $\mathbb{Z}_{>0}$ 到自身的双射，记 $y_n = x_{\varphi(n)}$，称数列 $\{ y_n \}$ 为 $\{ x_n \}$ 的一个重排，若 $\lim_{n \to \infty}x_n = a$，证明 $\lim_{n \to \infty}y_n = a$.

对于 $\{ x_n \}$，任取 $\varepsilon > 0$，有相应的 $n_\varepsilon$，对于有限集 $\{ x_{\varphi(1)}, \ldots ,x_{\varphi(n_\varepsilon-1)} \}$，其有最大值 $x_{\varphi(k)}$，则对于 $n > k$ 有 $|y_n - a| < \varepsilon$，故 $\{ y_n \}$ 亦收敛于 $a$.

### 3.2 数列极限的性质

> 3. 证明 $\lim_{n \to \infty} \frac{(2n-1)!!}{(2n)!!} = 0$.

\[ \begin{aligned}
 0 < \frac{(2n-1)!!}{(2n)!!} &= \prod_{k=1}^{n}\left(1-\frac{1}{2k}\right) < \prod_{k=1}^{n}\left(\exp(-\frac{1}{2k})\right) \\
 &= \exp\left( \sum^n_{k=1}(-\frac{1}{2k}) \right) < \exp\left( -\frac{1}{2}\ln n \right) = \frac{1}{\sqrt{n}}
\end{aligned} \]

（沃利斯乘积（Wallis product）：$\lim_{n \to \infty} \frac{(2n)!!}{(2n-1)!!} = \sqrt{\pi n}$）

> 11. 设函数 $f:\mathbb{R}\to \mathbb{R}_{\geq 0}$  满足：对任意的 $x,y\in \mathbb{R}$ 有
(1) $f(xy) = f(x)f(y)$;
(2) $f(x+y) \leq f(x)+f(y)$.
证明：$f(x+y) \leq \max(f(x),f(y))$ 对任意的 $x,y\in \mathbb{R}$ 均成立的充要条件是 $\forall n\in \mathbb{Z}_{>0},f(n)\leq 1.$

若 $f(1) = 0$ 命题显然成立，否则 $f(1)=1$，
（$\Rightarrow$） \[\forall n\in \mathbb{Z}_{>0}, f(n\cdot 1) \leq \max(\max(\cdots\max(f(1),f(1)),f(1)),f(1)) \leq 1.\]

（$\Leftarrow$）设 $f(x)\geq f(y)$ \[ \begin{aligned}
 f((x+y)^n) &= f\left( \sum^n_{k=0} \binom{n}{k}x^ky^{n-k} \right) \leq \sum^n_{k=0}f\left( \binom{n}{k}x^ky^{n-k} \right) \\
 &= \sum^n_{k=0}f\left( \binom{n}{k}\right)f(x^k)f(y^{n-k}) \\
 &\leq \sum^n_{k=0}1\cdot f(x^k)f(x^{n-k}) = (n+1)f(x^n)
\end{aligned} \]

故有 $f(x+y) \leq (1+n)^{\frac{1}{n}}f(x)$，其中 $(1+n)^{\frac{1}{n}} > 1$ 且 $\lim_{n \to \infty}(1+n)^{\frac{1}{n}} = 1$.

（满足题目条件的 $f$ 称为绝对赋值。对任意正整数 $n$，记 $\operatorname{ord}_p(n)$ 为 $n$ 的质因数分解中 $p$ 的重数，对于任意有理数 $m=\frac{a}{b}\in \mathbb{Q}$，定义 $m$ 的 p-进绝对赋值 $v_p(m) = \operatorname{ord}_p(a) - \operatorname{ord}_p(b)$.
奥斯特洛夫斯基定理（Ostrowski's theorem）：$\mathbb{Q}$ 上非平凡的绝对赋值或为通常的绝对值（$\exist n\in \mathbb{Z},f(n)>1$），或为 p-进绝对赋值（$ \forall n\in \mathbb{Z},f(n)\leq 1$）。）

### 3.3 单调有界收敛原理

> 18. （费克特次可加性引理（Fekete's subadditive lemma））设数列 $\{ x_n \}$ 满足 $x_{m+n} \leq x_m + x_n$，并且 $\{ \frac{x_n}{n} \}$ 有下界，证明 $\{ \frac{x_n}{n} \}$ 收敛。

对任意 $p\in \mathbb{Z}_{>0}$，将 $n\in \mathbb{Z} > p$ 用带余除法表示为 $n = qp + r$，则
\[ x_n \leq qx_p + x_r,\quad \frac{x_n}{n} \leq \frac{n-r}{n}\cdot \frac{x_p}{p} + \frac{x_r}{n}, \]

其中 $x_r \leq \max(x_1,x_2, \ldots ,x_{p-1})$，因而 \[ \limsup_{n\rightarrow \infty}\frac{x_n}{n} \leq \frac{x_p}{p}; \] 

又由确界原理，$\{ \frac{x_n}{n} \}$ 有下确界，故 \[ \inf \frac{x_n}{n} \leq \liminf_{n\rightarrow \infty}\frac{x_n}{n} \leq \limsup_{n\rightarrow \infty}\frac{x_n}{n} \leq \inf_{p\in \mathbb{Z}_{>0}} \frac{x_p}{p} = \inf \frac{x_n}{n}, \] 由夹逼定理即得。

> 19. 利用闭区间套定理证明确界原理。

对 $\mathbb{R}$ 的任意非空子集 $A$，假定 $b_0$ 为其上界，任取 $a_0\in A$，若 $\frac{a_i+b_i}{2} \in A$，取 $a_{i+1} = \frac{a_i+b_i}{2}, b_{i+1} = b_i$，反之则取 $a_{i+1} = a_i, b_{i+1} = \frac{a_i+b_i}{2}$，易知闭区间族 $[a_i,b_i]$ 满足闭区间套定理条件，且 $\xi = \bigcap^{\infty}_{i \in  I}[a_i,b_i]$ 为最小上界。

> 20. （1）证明任一数列必有单调子列；（2）利用单调有界收敛原理证明波尔查诺-魏尔斯特拉斯定理。

对任意数列 $S_0$，取其首项 $s_0$，将 $S_0$ 按与 $s_0$ 的相对大小分为三个子集 $L_0 \sqcup M_0 \sqcup U_0$，若 $M_0$ 为无限集则其即为单调子列，否则 $L_0,U_0$ 至少有一为无限集，记为 $S_1$，迭代上述操作。若操作在 $M_i$ 处中断则其即为单调子列，否则得到序列 $\{ s_0,s_1, \ldots  \}$，对任意 $s_i$，若其对应操作为 $S_{i+1} = L_{i}$，则 $\forall k>i, s_k \in L_i < s_i$，反之亦然。序列各项对应操作中，$L$ 与 $U$ 至少有一类被选择了无限次，其对应 $s$ 即构成单调递减或递增序列。

（上述操作中两次“至少有一”的判定与下列定理的证明类似，最终都是（无限）鸽笼原理的应用：
柯尼格引理（König's lemma）：给定具有无穷个顶点但每个顶点的度有限的连通图 $G$，则对 $G$ 的任意顶点都至少存在一条无穷的简单路径。
无限拉姆齐定理（Infinite Ramsey Theorem）：如果对完全图 $K_\mathbb{N}$​ 的边用两种颜色进行染色，则一定存在一个无限大的子集，其中所有边都是同一种颜色的。）

> 21. 利用波尔查诺-魏尔斯特拉斯定理证明单调有界收敛原理。

不失一般地，假定有界数列 $\{ x_n \}$ 单调递增，由有界知其有一收敛子列 $\{ x_{\varphi(n)} \}$，假定其极限为 $a$. 对原数列任意一项 $x_i$，必然存在 $\varphi(j) < i < \varphi(k)$，否则 $\{\varphi(n)\}$ 至多只有 $i-1$ 项，由单调性知 $|n_{\varphi(j)}-a| < |n_i - a| < |n_{\varphi(k)}-a|$.

> 22. 利用戴德金定理证明波尔查诺-魏尔斯特拉斯定理。

假定数列 $\{ x_n \}$ 有上下界 $a,b$，定义 $\mathbb{R}$ 的子集 $U = \{ u: |(-\infty,u) \cap \{ x_n \}| = |\mathbb{N}| \}$, $L = \{ l:\exist n\in \mathbb{Z},|(-\infty,l)\cap\{ x_n \}| = n \}$，可知两集合符合戴德金定理条件，于是存在 $\xi$ 满足 $L \leq \xi \leq U$.
$\forall \varepsilon > 0, \xi + \varepsilon \in U$，故 $|[\xi,\xi+\varepsilon)\cap\{ x_n \}| = |\mathbb{N}|$，因而存在子列收敛于 $\xi$.

> 23. 利用闭区间套定理证明有限覆盖定理。

设 $(I_\lambda)_{\lambda\in L}$ 是 $[a_0,b_0]$ 的一族开覆盖，反设 $[a_0,b_0]$ 没有有限子覆盖，则 $[a_0,\frac{a_0+b_0}{2}],[\frac{a_0+b_0}{2},b_0]$ 至少有一没有有限子覆盖，记其为 $[a_1,b_1]$，并迭代上述操作；可知闭区间族 $[a_i,b_i]$ 满足闭区间套定理条件，而 $\xi = \bigcap^{\infty}_{i \in \mathbb{Z}}[a_i,b_i]$ 一定被某一开区间 $I_\gamma$ 覆盖，显然有无限组 $[a_i,b_i] \subseteq I_\gamma$，矛盾。

> 24. 利用有限覆盖定理证明波尔查诺-魏尔斯特拉斯定理。

假定数列 $\{ x_n \}$ 有上下界 $a,b$. 反设 $\{ x_n \}$ 没有收敛子列，则对任意 $\xi \in [a,b]$，存在 $\varepsilon$-邻域只含有限项 $x_n$. 因而定义 $I_\lambda:\exist \lambda\in \mathbb{Z},|I\cap\{ x_n \}| = \lambda$，由反证假设知 $\{I_\lambda\}_{\lambda\in \mathbb{Z}}$ 构成 $[a,b]$ 的一族开覆盖，由有限覆盖定理知存在有限子覆盖，其中只包含有限项 $x_n$，与 $\{ x_n \}$ 为无限集矛盾。

### 3.4 Cauchy 收敛准则

> 3. 设 $x_1 > 0$, $x_{n+1} = \frac{c(1+x_n)}{c+x_n}$（$c>1$），证明 $\{ x_n \}$ 收敛并求其极限。

引入齐次坐标，原递推关系式即 $(x_{n+1},1) = (c(1+x_n),c+x_n)$，亦即 \[ \begin{pmatrix} 
    x_{n+1} \\ 1  
\end{pmatrix} = \begin{pmatrix} 
    c & c \\ 1 & c 
\end{pmatrix} \begin{pmatrix} 
    x_n \\ 1 
\end{pmatrix}, \] 上述递推矩阵特征值为 $c\pm \sqrt{c}$，主特征值为 $c+\sqrt{c}$；矩阵的幂次会收敛至主特征值方向： $\lim_{n \to \infty}M^nA = \lambda_k^n \mathbf{v}_k$（LADW p.115）. 解特征向量： \[ \begin{pmatrix} 
    -\sqrt{c} & c \\ 1 & -\sqrt{c} 
\end{pmatrix} \begin{pmatrix} 
    u \\ v 
\end{pmatrix} = 0 \] 即 $u - \sqrt{c}v = 0, \lim_{n \to \infty}x_n = \frac{u}{v} = \sqrt{c}$.

> 6. （压缩映像原理）设 $I$ 是一个闭区间，$f:I \to I$，我们记 $f_1 = f$ 以及 $f_{n+1} = f\circ f_n$. 现设存在正整数 $k$ 和常数 $\theta \in (0,1)$ 使得对任意的 $x,y\in I$ 有 \[ \left\lvert f_k(x) - f_k(y) \right\rvert \leq \theta \left\lvert x-y \right\rvert, \] 证明存在唯一的 $a\in I$ 使得 $f(a)=a$. 

任取 $x\in I$，对数列 $\{ f_i(x) \}$，任取 $m>n>k$，记 $p = \left\lfloor \frac{n}{k} \right\rfloor$：\[\begin{aligned}
 |f_m(x) - f_n(x)| &= |f_k(f_{m-k}(x)) - f_k(f_{n-k}(x))| \\
 &\leq \theta|f_{m-k}(x) - f_{n-k}(x)| = \cdots \\
 &\leq \theta^p |f_{m-pk}(x) - f_{n-pk}(x)| \leq \theta^p 2d(I),
\end{aligned} \] 易知 $\{ f_i(x) \}$ 为柯西列，记其收敛于 $a$，对 $f_{n+1}(x) = f\circ f_n(x)$ 取极限得 $a = f(a)$. 假定另有 $\{ f_i(y) \}$ 收敛于 $b$，易知 $\{f_i(x) - f_i(y)\}$ 收敛于 $0$，故 $b=a$.

### 3.5 数列的上极限和下极限

> 3. 设 $\{ x_n \},\{ y_n \}$ 均是有界数列，且 $x_n \geq 0, y_n \geq 0$. 证明：\[
 \varliminf_{n\to \infty}x_n + \varliminf_{n\to \infty}y_n \leq \varliminf_{n\to \infty}(x_n + y_n) \leq \varliminf_{n\to \infty}x_n + \varlimsup_{n\to \infty}y_n \\
 \varliminf_{n\to \infty}x_n + \varlimsup_{n\to \infty}y_n \leq \varlimsup_{n\to \infty}(x_n+y_n) \leq \varlimsup_{n\to \infty}x_n + \varlimsup_{n\to \infty}y_n
\]
> 4. 设 $\{ x_n \},\{ y_n \}$ 均是有界数列，且 $x_n \geq 0, y_n \geq 0$. 证明：\[
 \varliminf_{n\to \infty}x_n\varliminf_{n\to \infty}y_n \leq \varliminf_{n\to \infty}(x_n y_n) \leq \varliminf_{n\to \infty}x_n\varlimsup_{n\to \infty}y_n \\
 \varliminf_{n\to \infty}x_n\varlimsup_{n\to \infty}y_n \leq \varlimsup_{n\to \infty}(x_ny_n) \leq \varlimsup_{n\to \infty}x_n\varlimsup_{n\to \infty}y_n
\]

设 $\varliminf_{n\to \infty}x_n = a$，则存在子列 $\{ x_{n_i} \}$ 收敛于 $a$，对于子列 $\{ y_{n_i} \}$，存在子列 $\{ y_{n_{i_j}} \}$ 收敛于 $b = \varlimsup_{n\to \infty}y_{n_i}$，则 $\varliminf_{n\to \infty}(x_n y_n) \leq ab \leq \varliminf_{n\to \infty}x_n\varlimsup_{n\to \infty}y_n$. 其余类似。

> （法图引理（Fatou's lemma））测度空间 $(S,\Sigma,\mu)$ 上的正实值可测函数列 $\{f_n\}$ 满足： \[
 \int _{S}\varliminf _{n\to \infty }f_{n}\,\mathrm{d}\mu \leq \varliminf _{n\to \infty }\int _{S}f_{n}\,\mathrm{d}\mu
\]

应用到自然数上的计数测度即得：对于正值数列，下极限的加法可以拓展至无限求和，而对于上极限，在无限求和情况下变号。
经典的例子是 $x^{(n)}_k = \delta_{nk}$，则 \[
 \varlimsup_{n\to \infty}\sum_k x^{(n)}_k = 1,\quad \sum_k\varlimsup_{n\to \infty} x^{(n)}_k = 0.
\] 不对称性来自于正值的要求，这固定了一个下界，由单调有界收敛原理即可证明上述法图引理；类似地，如果固定上界，则有反向法图引理。
重新从戴德金分割的角度看，由可数集合的上下极限，3. 一式第一个 $\leq$ 与 4. 二式第二个 $\leq$ 即：
\[
 \bigcup^{\infty}_{i=1}\bigcap^{\infty}_{j=i}  x_j \cup \bigcup^{\infty}_{i=1}\bigcap^{\infty}_{j=i}  y_j \subseteq \bigcup^{\infty}_{i=1}\bigcap^{\infty}_{j=i} (x_j\cup y_j) \\
 \bigcap^{\infty}_{i=1}\bigcup^{\infty}_{j=i} (x_j\cup y_j) \subseteq \bigcap^{\infty}_{i=1}\bigcup^{\infty}_{j=i}  x_j \cup \bigcap^{\infty}_{i=1}\bigcup^{\infty}_{j=i}  y_j
\] 

> 7. 设 $\{ x_n \}$ 是一个正数列，证明 \[ \varliminf_{n\to \infty}\frac{x_{n+1}}{x_n} \leq \varliminf_{n\to \infty}\sqrt[n]{x_n} \leq \varlimsup_{n\to \infty}\sqrt[n]{x_n} \leq \varlimsup_{n\to \infty}\frac{x_{n+1}}{x_n}. \] 

考虑第一个 $\leq$. 若 $\varliminf_{n\to \infty}\frac{x_{n+1}}{x_n} = 0$，由 $\sqrt[n]{x_n} > 0$，显然成立；否则设 $\varliminf_{n\to \infty}\frac{x_{n+1}}{x_n} = a$，即 \[
 \forall a > \varepsilon > 0\,\exist n_\varepsilon \, \forall n\geq n_\varepsilon, \frac{x_{n+1}}{x_n} > a-\varepsilon \\
 \frac{x_{n+1}}{x_{n_\varepsilon}} = \frac{x_{n+1}}{x_n}\frac{x_n}{x_{n-1}}\ldots\frac{x_{n_\varepsilon + 1}}{x_{n_\varepsilon}} > (a-\varepsilon)^{n-n_\varepsilon +1},
\] 开 $n$ 次方并取 $n\to \infty$ 得 \[
 \varliminf_{n\to \infty}\sqrt[n]{x_n} \geq \varliminf_{n\to \infty}(a-\varepsilon)\sqrt[n]{x_{n_\varepsilon}(a-\varepsilon)^{1-n_\varepsilon}} = (a-\varepsilon)\cdot 1，
\] 由 $\varepsilon$ 的任意性即得。

> 9. 假设对任意的正整数 $n$ 均有 $x_n \geq 0$, 证明 $\varlimsup_{n\to \infty}\sqrt[n]{x_n} \leq 1$ 的充要条件是：对任意的实数 $\ell > 1$ 均有 $\lim_{n \to \infty}\frac{x_n}{\ell^n} = 0.$

（$\Rightarrow$） \[
 \forall \ell>1\,\exist n_\ell\,\forall n>n_\ell, \frac{x_n}{\ell^n} = (\frac{\sqrt[n]{x_n}}{\ell})^n \leq (\frac{\frac{1+\ell}{2}}{\ell})^n,
\] 取极限即得。
（$\Leftarrow$） \[
 \forall \varepsilon > 0\,\exist n_\varepsilon\,\forall n \geq n_\varepsilon, \frac{x_n}{\ell^n} < \varepsilon, \\
 x_n < \varepsilon\ell^n, \sqrt[n]{x_n} < \ell\sqrt[n]{\varepsilon}, \varlimsup_{n\to \infty}\sqrt[n]{x_n} \leq \ell,
\] 由 $\ell$ 的任意性即得。

> 10. 设 $\{ x_n \}$ 为正数列，
>> (1) 证明 \[
 \varlimsup_{n\to\infty} n\left( \frac{1+x_{n+1}}{x_n} -1 \right) \geq 1;
\] 

反设存在 $\{ x_n \}$ 使得 \[
 \exist\varepsilon>0\,\exist N\,\forall n\geq N, n\left( \frac{1+x_{n+1}}{x_n} -1 \right) < 1-\varepsilon < 1, \\
 \frac{x_n}{n} - \frac{x_{n+1}}{n+1} > \frac{1}{n+1}, \\
 \frac{x_N}{N} - \frac{x_n}{n} > \frac{1}{N+1} + \frac{1}{N+2} + \cdots + \frac{1}{n};
\] 取极限，右式发散，故矛盾。

>> (2) 证明 (1) 中的下界是最优的，即存在正数列 $\{ x_n \}$ 取等。

取 $x_n = n\ln n$，则 \[
 \lim_{n \to \infty} n\left( \frac{1+x_{n+1}}{x_n} -1 \right) = \lim_{n \to \infty} \frac{1+\ln\left( (\frac{n+1}{n})^n \right) + \ln(n+1)}{\ln n} = 1,
\] 故 $\varlimsup_{n\to\infty} n\left( \frac{1+x_{n+1}}{x_n} -1 \right) = 1$.

> 12. 设有界数列 $\{ x_n \}$ 发散，记 \[
 h = \varliminf_{n \to \infty} x_n, \quad H = \varlimsup_{n \to \infty} x_n.
\] 证明：若 $\varliminf_{n \to \infty} (x_n - x_{n-1}) = 0$ 或 $\varlimsup_{n \to \infty} (x_n - x_{n-1}) = 0$，则集合 $\{ x_n \mid n \in \mathbb{Z}_{>0} \}$ 在区间 $[h,H]$ 稠密。

反设存在开区间 $A_\varepsilon = (a-\varepsilon, a+\varepsilon) \subseteq [h,H]$ 满足 $\{ x_n \}\cap A_\varepsilon = \empty.$ 记 \[
 L = [h,a-\varepsilon]\cap\{ x_n \},\quad U = [a+\varepsilon,H]\cap \{ x_n \}.
\] 由上下极限定义知 $L, U$ 均为无穷集，故对 $m \in L, n \in U,$ 存在无限对 $m < n$ 与 $m > n.$
对于每一对 $m < n,$ 均存在 $m\leq k < k+1 \leq n$ 使得 $a_{k+1} - a_k \geq 2\varepsilon$，于是 $\varlimsup_{n \to \infty} (x_n - x_{n-1}) > 0$，反之则有 $\varliminf_{n \to \infty} (x_n - x_{n-1}) < 0.$

> 13. 证明集合 $\{ \ln n -\lfloor \ln n \rfloor \mid n \in \mathbb{Z}_{>0} \}$ 在 $[0,1]$ 中稠密。

定义数列 $\{ x_n \} = \{ \ln n - \lfloor \ln n \rfloor \},$，显然其有界且发散。
考虑子列 $\{ x_{\lceil e^n \rceil } \}$，由 $e^n \leq \lceil e^n \rceil < e^n + 1$，取对数得 \[
 n \leq \ln(\lceil e^n \rceil ) < \ln(e^n+1) = n + \ln(1 + e^{-n}).
\] 易知该子列收敛至 $0$，因而有 $\varliminf_{n \to \infty} x_n = 0.$ 类似地可得 $\varlimsup_{n \to \infty} x_n = 1.$
定义数列 $ \{ y_n \} = \{ x_{n+1} - x_n \} = \{ \ln(\frac{n+1}{n}) + \lfloor \ln(n+1) \rfloor - \lfloor \ln n \rfloor \},$ 由 $\ln n$ 发散且 $\ln(\frac{n+1}{n}) \leq \ln 2 < 1$ 知 $\lfloor \ln(n+1) \rfloor - \lfloor \ln n \rfloor$ 项为 $1$ 或 $0$ 且两者均出现无限次，因而 $\varlimsup_{n \to \infty} y_n = 1, \varliminf_{n \to \infty} y_n = 0.$
由 12. 即得。
（前述 $n \leq \ln(\lceil e^n \rceil )$ 一步可加强为严格小于，这里要求的 $e^n$ 为无理数（$n$ 可加强为任意非零有理数）有远较 $e$ 的超越性更加简明的[证明](https://proofwiki.org/wiki/Exponential_of_Rational_Number_is_Irrational)。）

### 3.6 Stolz 定理

> 6. 设 $k>0, x_1 \in (0,k),$ $x_{n+1} = x_n (1 - \frac{x_n}{k}), \forall n \in \mathbb{Z}_{>0},$ 证明 $\lim_{n \to \infty} n x_n = k.$

令 $y_n = \frac{x_n}{k},$ 即 $y_{n+1} = y_n (1 - y_n),$ 归纳可得 $y_{n+1} < y_n,$ 由单调有界收敛原理知 $y_n$ 收敛至 $a,$ 代入递推式解得 $a = 0.$
由施托尔茨定理，\[
 \lim_{n \to \infty} n y_n = \lim_{n \to \infty} \frac{n}{y_n^{-1}} = \lim_{n \to \infty} \frac{1}{y_{n+1}^{-1} - y_n^{-1}} = \lim_{n \to \infty} (1-y_n) = 1.
\] 

（对于标准逻辑斯谛映射 $x_{n+1} = r x_n (1 - x_n)$, $r \in (0,1)$ 时 $x_n$ 指数衰减，$r = 1$ 时幂律衰减，$r > 1$ 时收敛至不动点。）

> 7. 设 $k$ 为正整数，且正数列 $\{ x_n \}$ 满足 $\lim_{n \to \infty} x_n (x_1^k + \cdots + x_n^k) = 1.$ 证明 $\lim_{n \to \infty} n x_n^{k + 1} = \frac{1}{k + 1}.$

记 $S_n = x_1^k + \cdots + x_n^k.$ 可证 $x_n$ 收敛于 $0$ 而 $S_n$ 发散于正无穷。
\[\begin{aligned}
 \lim_{n \to \infty} n x_n^{k+1} &= \lim_{n \to \infty} \frac{n}{S_n^{k+1}} = \lim_{n \to \infty} \frac{n + 1 - n}{(S_n + x_{n+1}^k)^{k+1} - S_n^{k+1}} \\
 &= \lim_{n \to \infty} \frac{1}{\sum^n_{i=1}\binom{k+1}{i}S_n^{k+1-i} x_{n+1}^{k+i}} \\
 &= \lim_{n \to \infty} \frac{1}{(k + 1)S_n^{k} x_{n + 1}^k + \cdots } \\
 &= \lim_{n \to \infty} \frac{1}{(k+1)\left( \frac{S_n}{S_{n+1}}(S_{n+1}x_{n+1}) \right)^k } \\
 &= \lim_{n \to \infty} \frac{1}{(k+1)\left( \frac{x_1^k + \cdots + x_n^k}{x_1^k + \cdots + x_n^k + x_{n+1}^k} \right)^k } = \frac{1}{k+1}.
\end{aligned}\] 

> 9. （柯西乘积）设 $\lim_{n \to \infty} x_n = a, \lim_{n \to \infty} y_n = b,$ 证明 \[
 \lim_{n \to \infty} \frac{x_1 y_n + x_2 y_{n-1} + \cdots + x_n y_1}{n} = ab.
\] 

由题设，$\forall \varepsilon > 0\,\exist N\,\forall n > N, |x_n - a| < \varepsilon, |y_n - b| < \varepsilon.$ 对于大于 $2N$ 的 $n$ 有： \[
 \frac{1}{n}\sum^n_{i=1} x_i y_{n+1-i} = \frac{1}{n}\sum^{N}_{i=1} (x_i y_{n+1-i} + x_{n+1-i} y_{i}) + \frac{1}{n}\sum^{n-N}_{i = N+1} x_i y_{n+1-i},
\] 右式第一个求和中，所有含 $n$ 项均属于 $a$ 或 $b$ 的 $\varepsilon$-邻域，且求和项共有常数 $N$ 项，第二个求和中所有含 $n$ 项均属于 $a$ 或 $b$ 的 $\varepsilon$-邻域；取 $n$ 大于第一个求和与 $\varepsilon$ 之比即可。

（类似地可以证明，若 $\sum x_n$ 与 $\sum y_n$ 两者均强切萨罗可和（绝对偏差部分和收敛），或其一绝对切萨罗可和（差项绝对值部分和收敛）、其一切萨罗可和，则柯西乘积切萨罗可和；前者限制了不在 $a,b$ 邻域的项的密度，后者限制了总体波动的能量大小。）
